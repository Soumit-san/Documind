## Phase 1: MVP Features

### F-01: Auto Document Detection
- **Frontend (Extension)**
  - [ ] Implement background service worker to detect supported document URLs or MIME types (PDF, DOCX, etc.).
  - [ ] Trigger Chrome extension sidebar injection upon detection.
  - [ ] Build toggle logic via extension icon click (Alt+D shortcut).
  - [ ] Implement 380px sidebar UI with neo-brutalist styling.

### F-02: Instant Document Summarization
- **Backend**
  - [ ] Create API endpoint `POST /api/summarize`.
  - [ ] Implement document text extraction (PyMuPDF, etc.).
  - [ ] Implement tiered summarization logic (Executive, Section-by-Section, Key Entities).
  - [ ] Ensure inline citations are generated by the LLM.
- **Frontend**
  - [ ] Build Summary tab UI in the sidebar.
  - [ ] Add loading state (neo-brutalist progress indicator).
  - [ ] Display tiered summary with clickable citation chips.

### F-03: Conversational Q&A (RAG Engine)
- **Backend**
  - [ ] Create API endpoint `POST /api/chat`.
  - [ ] Implement document chunking (512 tokens, 64 overlap) and embeddings (local model).
  - [ ] Upsert vectors to ChromaDB.
  - [ ] Implement semantic search and context retrieval.
  - [ ] Connect to LLM (Ollama/Gemini/Groq) for answer generation with citations.
  - [ ] Implement fallback behavior if the answer is not found.
- **Frontend**
  - [ ] Build Chat tab UI in the sidebar and Web App.
  - [ ] Implement message bubbles, user/assistant distinction, and typing indicator.
  - [ ] Implement follow-up question suggestions.

### F-04: Multi-Document Corpus (Web App)
- **Backend**
  - [ ] Create API endpoints for multi-file upload, document management (folders/projects).
  - [ ] Create persistent vector index per user.
  - [ ] Modify RAG engine to query across multiple documents within a folder.
- **Frontend (Web App)**
  - [ ] Build drag-and-drop file upload interface.
  - [ ] Build library view for organizing documents.
  - [ ] Implement multi-document chat interface.

### F-05: Export & Share
- **Backend**
  - [ ] Create API endpoints for generating PDF/DOCX from summaries.
  - [ ] Implement link-sharing functionality with optional password protection.
- **Frontend**
  - [ ] Build Export tab UI with "Copy to clipboard" and file download options.
  - [ ] Add share button and password configuration modal in Web App.
